# Remove my profile picture

### User Story

**As a** user,
**I want to** remove my current profile picture,
**So that** I can revert to a default avatar or placeholder.

---

### Acceptance Criteria
*This list defines what must be true from the user's perspective for the story to be considered "done."*

- [ ] On my profile page, there is an option to remove my current picture.
- [ ] When I confirm the removal, my custom picture is deleted.
- [ ] My profile now displays a default avatar or placeholder image.
- [ ] The change is reflected throughout the application where my picture was previously shown.
