# User Profile Management

### User Story

**As a** registered user,
**I want to** manage my profile information,
**So that** I can keep my personal details up-to-date and personalize my account.

---

### Acceptance Criteria
*This list defines what must be true from the user's perspective for the story to be considered "done."*

- [ ] A dedicated profile management page is accessible to logged-in users.
- [ ] Users can view their current profile information (e.g., name, email, profile picture).
- [ ] Users can modify their profile information.
- [ ] Changes to the profile are saved and reflected throughout the application.
