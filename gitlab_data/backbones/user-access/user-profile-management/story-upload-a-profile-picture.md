# Upload a profile picture

### User Story

**As a** user,
**I want to** upload a picture for my profile,
**So that** other users can recognize me and my profile feels personalized.

---

### Acceptance Criteria
*This list defines what must be true from the user's perspective for the story to be considered "done."*

- [ ] On my profile page, there is an option to upload a new picture.
- [ ] I can select an image file (e.g., JPG, PNG) from my device.
- [ ] After selecting a file, the upload process begins and I see a confirmation when it is successful.
- [ ] My new picture is displayed on my profile page and in other relevant places (e.g., navigation bar, comments).
- [ ] The system provides clear feedback if the upload fails (e.g., wrong file type, file too large).
