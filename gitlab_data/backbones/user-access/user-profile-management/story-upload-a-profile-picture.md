# Upload a profile picture

### User Story

**As a** registered user,
**I want to** upload an image to use as my profile picture,
**So that** I can personalize my account and be easily recognized by others.

---

### Acceptance Criteria
*This list defines what must be true from the user's perspective for the story to be considered "done."

- [ ] A user can find an 'Upload Picture' or 'Change Picture' option on their profile page.
- [ ] When the upload option is selected, the user can choose a standard image file (e.g., JPG, PNG) from their device.
- [ ] After a successful upload, the new picture immediately replaces the old one (or the default avatar) on the profile page.
- [ ] The user's new profile picture is visible in all relevant areas of the application (e.g., next to comments, in user lists).
- [ ] The user receives a clear success confirmation message after the upload is complete.
- [ ] If the upload fails (e.g., invalid file type, file too large), the user receives a clear error message explaining the problem.
